# Security Policy

FreeBank is experimental, pre-audit software for test networks only.
Do not use it with anything of real value.

## Reporting a vulnerability

Please email **mbdcdev@gmail.com** rather than opening a public issue,
so problems can be fixed before they are broadcast. Include steps to
reproduce if you can. You should hear back within a few days.

Highest-value reports: anything touching consensus, the two-way peg
(deposits / withdrawal bundles / BMM), reserve accounting, or wallet
key handling.

Ordinary bugs (crashes, RPC misbehaviour, build problems) are welcome
as public [issues](../../issues).
